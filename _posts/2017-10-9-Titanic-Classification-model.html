Titanic Classification Kaggle Competition¶

Data Dictionary¶


Dependent Variable¶

**Survived:** Categorical variable
expresses if the titanic passenger survived or perished 1 indicates they survived 0 indicates they perished

Independent Variables¶

**Age:** Numeric variable
that represents the age of the passenger
**Cabin:** Categorical variable
indicating the part of the ship that they were styaing in
**Embarked:** Categorical variable
representing which port the passenger got on the titanic
**Fare:** Numeric variable
The amount of money the passenger used to purchase their ticket
**Name:** Categorical variable
The name of the passenger
**Parch:** Numeric variable
The amount of parents and children they have aboard the titanic
**PassengerId:** Categorical variable
The ID number assigned and unique to each passenger
**Pclass:** Categorical variable
the economic class of each passenger 1 being the highest 3 being the lowest
**Sex:** Categorical variable
The gender of each passenger
**SibSp:** Categorical variable
The amount of siblings and spouses the passenger has aboard the titanic
**Ticket:** Categorical variable
The ticket ID for each ticket bought

rown=0
# iterate using an indexer over every row in dataframe ship
for index, row in ship.iterrows():
    # if deck is a null value continue to the next row
    if(pd.isnull(row['Deck'])):
        continue
    else:
        # If deck is not a null value check if the passenger has relatives aboard the ship
        if row['SibSp']>0 or row['Parch']>0:
            # If they do store their last name and deck in variables 
            name=row['Lname']
            deck=row['Deck']
            # Now begin looping through the Dataframe again looking for passengers with similar last names
            for index2, row in ship.iterrows():
                if row['Lname']==name:
                    if(pd.isnull(row['Deck'])):
                    # Once you find relatives and they have a null value in deck assign them the same deck 
                        ship.at[index2,'Deck']=deck
    rown+=1
