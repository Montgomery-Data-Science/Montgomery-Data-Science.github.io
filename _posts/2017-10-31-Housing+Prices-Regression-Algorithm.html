Step 1: Understand the Business Question¶

This regression model and those like it that predict house prices can be used in conjunction with real estate businesses to better understand the underlying worth of a home. Consumers can also use this tool to better understand what features of a house most factor in to increased prices.

Step 2: Data Understanding¶

This data set was collected from real homes in Ames Iowa, and was taken from the kaggle website. The goal of this algorithm is to use the train data set to create and fit a model that will then predict the house prices for the test data set. This algorithm is a part of the kaggle competition for house prices, and thus the test dataset does not include house prices. Thus, we must use one data set to predict house prices for the other data set.
